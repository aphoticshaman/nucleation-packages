## Usage Guidelines

### Color Palette Alignment
All assets are designed for dark mode with the LatticeForge color system:
- **Neural/Cyan**: `#06b6d4` - AI/ML features
- **Symbolic/Amber**: `#f59e0b` - Rule-based logic
- **Risk/Red**: `#ef4444` - Alerts, critical items
- **Background**: Slate-900 (`#0f172a`)

### File Format Notes
- Most images are PNG with transparency support
- Dimensions vary: 1024x1024 (Midjourney v6) or 1792x1024 (panoramic)
- JPG used for photographs (e.g., bluebedlam.jpg)

### Optimization Recommendations
1. Create WebP versions for production
2. Generate responsive srcsets (400w, 800w, 1200w)
3. Use lazy loading for below-fold images
4. Consider CSS blur placeholders for large images
