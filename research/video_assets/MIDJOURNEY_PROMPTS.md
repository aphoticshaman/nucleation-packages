## üìê PARAMETERS GUIDE

- `--ar 16:9` = Video aspect ratio (1920x1080)
- `--ar 1:1` = Square (logos, icons)
- `--v 6.1` = Latest Midjourney version
- Add `--style raw` for less stylized/more realistic
- Add `--chaos 20` for more variation
- Add `--q 2` for higher quality (uses more GPU)

## üìπ VIDEO EDITING TIPS

1. **Ken Burns effect** - Slow zoom on static images
2. **Crossfade** - 0.5s transitions between sections
3. **Lower thirds** - Use charts with speaker overlay
4. **Motion blur** - Add to B-roll for dynamism
5. **Music** - Epic orchestral or modern electronic
