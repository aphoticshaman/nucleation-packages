## Visual Style Guide

**Color Palette:**
- Primary: Deep blue (#1a365d)
- Secondary: Gold/amber (#d69e2e)
- Accent: Cyan (#0bc5ea)
- Background: Dark (#1a202c) or Light (#f7fafc)
- Error/Warning: Coral (#fc8181)

**Typography:**
- Headers: Montserrat Bold
- Body: Source Sans Pro
- Code: Fira Code
- Captions: Source Sans Pro Italic

**Diagram Style:**
- Clean, minimal lines
- Consistent node shapes
- Subtle shadows for depth
- Icons where helpful
- Color-coded by concept type

**Charts:**
- Clear axis labels
- Legends when needed
- Minimal gridlines
- Consistent color usage
- Annotations for key points
