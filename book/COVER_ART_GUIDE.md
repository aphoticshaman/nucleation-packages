# Book Cover Art Guide: The Mathematics of Intelligence

## KDP Technical Specifications

### For a 500-600 Page Book (6×9 trim)

**Paperback Cover Dimensions:**
- Trim size: 6" × 9" (standard for non-fiction, fits most content)
- Page count: ~550 pages (estimate)
- Paper type: Cream (easier on eyes for text-heavy)
- Spine width calculation: 550 × 0.0025" = **1.375"** spine

**Full Cover Dimensions (wrap-around):**
```
Width: Back cover (6") + Spine (1.375") + Front cover (6") + Bleed (0.25") = 13.625"
Height: 9" + Bleed (0.25") = 9.25"

Full cover: 13.625" × 9.25" at 300 DPI = 4088 × 2775 pixels
```

**eBook Cover:**
- Minimum: 1000 × 1600 pixels (2:3 ratio)
- Recommended: 1600 × 2400 pixels
- Maximum: 2500 × 2500 pixels (10,000px on longest side)

**File Requirements:**
- Format: PDF for paperback, JPEG/TIFF for eBook
- Resolution: 300 DPI minimum
- Color space: CMYK for print, RGB for eBook
- File size: Under 40MB for fast proofing

**Use the official calculator:** https://kdp.amazon.com/cover-calculator

## Cover Design Strategy

**Book Title:** The Mathematics of Intelligence
**Subtitle:** From Attention to AGI: A Practitioner's Guide to LLMs
**Author:** Ryan J. Cardwell
**Edition:** First Edition

### Visual Theme Options

**Option A: Abstract Neural/Mathematical**
- Interconnected nodes and paths (neural network aesthetic)
- Mathematical notation as visual elements
- Color: Deep blue/purple gradient with gold accents
- Style: Clean, sophisticated, slightly mysterious

**Option B: Emergence/Phase Transition**
- Crystalline structure forming from chaos
- Visual metaphor for order emerging from disorder
- Color: Dark background with luminous structures
- Style: Scientific, epic, transformative

**Option C: The Parrot That Dreams**
- Artistic interpretation of the book's central metaphor
- Surreal parrot with digital/mathematical elements
- Color: Vibrant but sophisticated
- Style: Memorable, conversation-starting, unique

**Option D: Cosmic/Universal**
- Brain/mind as cosmic structure
- Stars, galaxies, but also circuits and mathematics
- Color: Deep space with bright accents
- Style: Grand, philosophical, awe-inspiring

### Font Recommendations

- Primary: Bebas Neue, Oswald, or Montserrat (bold sans-serif)
- Alternative: Playfair Display (serif with gravitas)

**Subtitle:**
- Lighter weight of title font, or complementary serif
- Raleway Light, Source Sans Pro, or Lora

**Author Name:**
- Clean, readable, professional
- Match title family but lighter weight

**"First Edition":**
- Small caps, subtle
- Same family as author name

## Workflow

**Step 1: Generate Art**
- Run 3-4 Midjourney prompts
- Generate multiple variations of each (4 per prompt)
- Select top 3-5 candidates

**Step 2: Upscale Selected Art**
- Use Midjourney upscaler or Topaz Gigapixel
- Target: 4000×6000 pixels minimum for print
- Save as PNG (lossless)

**Step 3: Typography Overlay**
- Send me the selected art
- I'll create typography layouts with proper spacing, hierarchy
- Multiple versions for comparison

**Step 4: Assemble Full Cover**
- Create full wrap-around layout
- Add spine text and back cover elements
- Export: PDF (print), JPEG (eBook)

**Step 5: KDP Upload & Proof**
- Upload cover to KDP
- Order print proof ($$$)
- Check alignment, colors, spine readability
